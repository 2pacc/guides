## Project Convention

* Rails project 一律放在 ~/projects/project_name
* 一律使用 Pow 掛起 project。
* 本機重開 server，使用 `touch tmp/restart.txt`
* 拉下來的 code 如果不能動，先 `bundle install` -> rake db:migration -> `touch tmp/restart.txt`
